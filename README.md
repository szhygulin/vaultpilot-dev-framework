# vaultpilot-development-agents

Bugfixing and development agent framework for [vaultpilot-mcp](https://github.com/szhygulin/vaultpilot-mcp).

Agents poll GitHub issues and dispatch work based on labels (`bug`, `feature`, etc.):
each agent picks up an issue, implements a solution on a branch, and opens a PR
addressing it.

Early scaffolding. Not yet wired up.
